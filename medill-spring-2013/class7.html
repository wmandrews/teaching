Design for the story

Today we'll be exploring how to start from a base template of HTML and some story assets, and create a fully wireframed web-page.

  1. Create a project folder on your computer named design-story. I recommend putting it somewhere like your Documents folder.
  2. Download Bootstrap, unzip the file and drop the folders inside your new project folder. Take a look at the files inside.
  3. Go to the examples and click the Basic marketing site template. Save the page to design-story as template.html. Let's take a look at this file by opening in the browser and text editor.
  4. Explore the Bootstrap base CSS and components, and turn your earlier sketch into a wireframed page, making containers and layout for each of your items in the sketch. A wireframe is a type of sketch that typically shows how space will be allocated, but doesn't get into the details like color and specific design features within each piece in the wireframe. Ask me if you get stuck or would like some pointers.

    If you'd like to get more visual, changing colors, sizes, and adding actual visuals from the raw assets, I can show you some techniques. Just ask.
