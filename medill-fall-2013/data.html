Helpful shortcuts

Selecting a chunk of data, such as a row, column or the entire dataset is a common activity in spreadsheets. There are some handy keyboard shortcuts that cut down on the time it takes to do this. Below, CTRL is for PC, substitute the COMMAND key for Mac.

Jump to the next empty cell: CTRL+Arrow key in the direction you want to jump.

Select until the next empty cell: CTRL+SHIFT+Arrow key in the direction you want to select. This is often helpful when you want to select your entire dataset.

To do this, select the cell A1
and do CTRL+SHIFT+
then CTRL+SHIFT+ .

As long as there are no empty cells in your last column, you should have your entire table selected. You can keep doing the CTRL+SHIFT in the same direction to keep skipping to empty cells until you reach the bottom of your data.

Sort

Arranging your data in order from greatest to least, or vice versa, is important when preparing data for charting. Much charting software plots data points in the order they're entered, and it is easier to understand a list ordered by the plotted values.

Sorted

How to sort

Click a cell in the column you wish to sort. Make sure you don't have an entire column selected, or else your sort will corrupt the data by re-ordering only that column.

You can either click the sort icon in the toolbar, or select Data > Sort from the application menu for more advanced sorting options.

Filter

It is often important to look at subsets of your data. That's where the stories are often found. Here's one example in a graphic we did at the Post. Looking at tuition costs by a subset of schools types reveals dramatic differences in cost changes, though costs are increasing at every type.

How to filter

You can either click the filter icon in the toolbar, or select Data > Filter from the application menu.

Once you've activated the filter function, each of your header fields should have arrows next to them. To filter by a specific field, click its arrow and toggle the selections to get the subset you desire. This view has removed all of the "Not Voting" representatives. Excel displays a handy count of all the records found for a given filter. 423 representatives voted "Yea" or "Nay" in this vote.

Pivot tables

Large datasets can seem overwhelming. It is difficult to draw insight from scanning through hundreds of rows of data. However, if we group and transform that data, aggregating into useful categories and subsets, stories can start to emerge.

Excel provides a handy way to do this in the form of pivot tables. A pivot table allows you to group on a common field, like state or political party, and create summaries based on those groupings.

Download data for this activity

How to create a pivot table

Make sure you have selected either a single cell in your data or the entire dataset. Then select Data > PivotTable. Make sure the table or range that it's using looks correct, and click OK. You should then see something that looks like this:

Here's what each of the areas mean:

Report Filter: Works just like the regular filter function. Want to filter your data by a field? Drag it into this area.

Row Labels: These are the overall field(s) that you want to aggregate: i.e. if you want to see the data combined by state, drag the state field into this area.

Column Labels: Allow you to subdivide your rows into even further subsets. You could further divide the state-by-state breakdown by adding gender, to see values by state by gender.

Values: These are the field(s) that contain the values you'll be summarizing, often a numeric field. For example, if you have a list of all the crimes in D.C., you could perform a COUNT function on the crime type field, or an AVERAGE function on the perpetrator age field.
