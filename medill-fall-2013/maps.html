MAPS

MapBox

MapBox.com

This fast-growing mapping service is created and maintained by a D.C.-based company named Development Seed. They have a lot of different tools and libraries for creating maps.

After you've signed up for the service, you will have an admin interface accessible when you login to mapbox.com.

The admin

The main screen in mapbox will show you all the maps you've created, and allow you to start a new map.

Click NEW MAP to start a new map.

Create map

After clicking new map, you will be prompted to choose a base layer type. You'll want to think about the type of map you're creating and which type of base makes the most sense to show.

Presets

You are now viewing the map interface. The panels on the left are to edit the map display, and the right will show you what your map currently looks like. It's fully interactive.

These presets give several different color options for the MapBox Streets layer. Select a good one to start with for your map.

Layers

This panel gives you access to the visibility and colors of each of the main map layers: streets, buildings, areas (like parks), water and land, or terrain. You can adjust the hue (color), saturation, lightness and alpha (transparency) for each.

Markers

In this panel, you can add markers to the map. MapBox provides a good collection of icons for markers, and you can define any color you wish. The text fields will appear in a tooltip when a marker is selected. To add to the map, click ADD at the bottom of the window, then click anywhere on the map to place the icon.

Info

This is the panel for the overall info of the map, including title and description, and some options for different features like zoom controls and a geocoder (search for locations like you do in Google maps). When you're happy with your changes, you can click SAVE CHANGES.

It's very important to set an appropriate map origin for your map. Otherwise the map will always start at the middle of the earth view zoomed way out. If your map is of the D.C. region, you should zoom in to that area with your map appropriately centered and click SET underneath the map origin title.

Publish

After you've saved, a PUBLISH button should appear. This screen will show up after it's been clicked.

The simplest way to put a MapBox map on a page is to use the Easy Embed option. There you can select a height and width for your embedded map, and then copy the code below to paste into a website.

CartoDB

CartoDB.com

CartoDB is a similar service that has a more data-driven admin interface than MapBox. You can accomplish a lot of similar things using MapBox, but CartoDB allows you to accomplish more without writing any code.
