Week 7 assignment

Create an HTML page that uses jQuery to manipulate elements. Create buttons that change something on the page when you interact with them. It could be to show a hidden div when you hover over a button, or to add some text to the page when you click something.

You need to have at least 5 event handlers (interactions, like mouseover, mouseout, click) in your code, so you should have a few different html elements on your page that you can interact with. One item might have more than one interaction, and you can use the same interaction more than once (multiple elements can have a click event, for example).

As always, please include the necessary items in your head and add your own styles to the page.

Due in your week7 dropbox folder before class on March 15.
