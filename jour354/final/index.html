Main goal

Create a multimedia project that effectively tells a story that is relevant to the local area. The local area could be the college community, College Park, the DC metro area or even the state of Maryland.

Content

You will need to report and gather the content for this project. If you use video, photos or graphics that weren't created by you, make sure you have the permission of the creator to use in your project. The project should undergo all the necessary steps that you would follow to publish something on a media company's site. You will be rewarded the more you use original content in your project.

There is no set amount of video, text, photo or other element that you need to include, but your project must tell a compelling story using methods and techniques we've covered in this class. A sample project might have a main story page with a slideshow, some text and a graphic, along with additional pages that cover other parts of the story using more elements.

Please run your story idea by me before you do too much work on it. I can help give you ideas about types of content and can provide some resources that might make your reporting, data gathering and coding easier.

The point is, use the tools you've learned to make a story better than if it were just a text-based article.

HTML/CSS

The HTML and CSS should be laid out in a properly spaced format, with id's and classes being used to style the page elements. You should include comments in your code to note the different sections of your layout, along with any other comments you need to help you make your layout clear. The project should have a logical navigation. Don't forget title, keywords and a description in your page headers.

JavaScript

The JavaScript should be clean and easy to read, and should work without bugs. If it makes sense to load your data dynamically (does it change? is it a lot of data or media?), then you should use JavaScript. Otherwise, writing content directly in your HTML is fine. Include comments to explain what each chunk of code is supposed to do.

Organization

Make sure you organize your files in a logical way, according to the concepts we covered in class.

Local story ideas

A few ideas that might help you think of your own stories.

  • A guide to the Cherry Blossom Festival, including an event calendar, slideshow, and interactive map.
  • A profile of a local music venue or sports program (sports might be difficult to do much original reporting on).
  • An interactive game to tell a story. A good example of using a game to tell a story in a new way.
  • A resource of information for local government information and officials (profiles, publicly-available salaries, etc.).
  • A report on local restaurants and health codes (remember this?). You could do a simple sortable table with the data, or use something like Google Fusion Tables to create a map and table automatically.
  • An interactive timeline.
